# Fudan-Microelectronics-Courses

This repository contains the course materials for the courses offered by SME/ICMNE/FICS, Fudan University. The materials include lecture notes, assignments, and projects for various courses in the field of integrated circuits.

## Courses

- **[Embedded-Processor-and-System-on-Chip](./Embedded-Processor-and-System-on-Chip)**
  - `Assignments/`: Homework assignments and solutions.
  - `Labs/`: Lab exercises and projects.
  - `Notes/`: Lecture notes and reference materials.

- **[The-Fundamental-of-Computer-Software](./The-Fundamental-of-Computer-Software)**
  - Problems, exercises and practices.
